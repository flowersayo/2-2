
# 프로젝트 파일 목록

이 리포지토리에는 다음과 같은 주요 파일이 포함되어 있습니다:

1. [Generative Adversarial Network.ipynb](./Generative%20Adversarial%20Network.ipynb)
   - GAN(Generative Adversarial Network) 모델 구현과 관련된 코드입니다.

4. [Sentiment Analysis with Transformer.ipynb](./Sentiment%20Analysis%20with%20Transformer.ipynb)
   - Transformer 기반으로 감정 분석을 수행한 코드입니다.
