

# 선택 정렬
: 입력 배열 전체에서 최솟값을 선택하여 배열의 0번 원소와 자리를 바꾸고, 다음엔 0번 원소를 제외한 나머지 원소에서 최솟값을 선택하여, 배열의 1번 원소와 자리를 바꾼다.

# 의사코드

```
입력 : 크기가 n인 배열 A
출력 : 정렬된 배열 A

for i= 0 to n-2 //i : 최솟값이 위치할 자리
  min = i 
  for j = i+1 to n-1 // A[j] ~ A[n-1] 에서 최솟값을 찾는다.
    if A[j] < A[min]
      min = j // min은 최솟값이 존재하는 인덱스
  A[i] <-> A[min] //
      
```
